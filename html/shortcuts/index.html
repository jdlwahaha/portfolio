Useful Mac Keyboard Shortcuts

Useful folder navigation shortcuts

alt + cmd + i Get info window to see multiple files
alt + cmd +delete Delete files forever without going to trash
alt + cmd + Q Quit application but preserve history (e.g. last opened tabs)
alt + click taskbar icons (e.g wifi, volume) Show more detail menu options
alt + shift + brightness/volume Adjust finer screen brightness / keyboard brightness / volume

Quick open system preferences

alt + brightness adjustment (F1, F2) Open System Preference > Built-in Retina Display setting
alt + triple window (F3) Open System Preference > Mission Control
alt + keyboard brightness adjustment (F5, F6) Open System Preference > Keyboard
alt + volume adjustment (F10, F11, F12) Open System Preference > Sounds window directly
