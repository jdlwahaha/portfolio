Useful Mac Keyboard Shortcuts

Useful folder navigation shortcuts

⌥ + ⌘ + i Get info window to see multiple files
⌥ + ⌘ +delete Delete files forever without going to trash
⌥ + ⌘ + Q Quit application but preserve history (e.g. last opened tabs)
⌥ + click taskbar icons (e.g wifi, volume) Show more detail menu options
⌥ + ⇧ + brightness/volume Adjust finer screen brightness / keyboard brightness / volume

Quick open system preferences

⌥ + brightness adjustment (F1, F2) Open System Preference > Built-in Retina Display setting
⌥ + triple window (F3) Open System Preference > Mission Control
⌥ + keyboard brightness adjustment (F5, F6) Open System Preference > Keyboard
⌥ + volume adjustment (F10, F11, F12) Open System Preference > Sounds window directly

Note:
⌥ = option key
⌘ = command
⇧ = shift
